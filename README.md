# 🚀 MERN Stack Application

This is a full-stack web application built using the **MERN Stack**, which includes MongoDB, Express.js, React.js, and Node.js.

---

## 📌 What is MERN Stack?

**MERN** is a JavaScript-based technology stack used to build full-stack web applications.

MERN stands for:
- **MongoDB** – NoSQL database for storing data  
- **Express.js** – Backend web framework for building APIs  
- **React.js** – Frontend library for building user interfaces  
- **Node.js** – JavaScript runtime environment for running server-side code  

---

## 🏗️ MERN Stack Architecture

The application follows a client-server architecture:

- **Frontend (React.js):** Handles UI and user interactions  
- **Backend (Node.js + Express.js):** Handles business logic and APIs  
- **Database (MongoDB):** Stores application data  
- **API Communication:** Frontend communicates with backend using HTTP/REST APIs

## 🎨 Bootstrap

Bootstrap is used to create a **responsive and mobile-friendly user interface**.  
It provides pre-built components like buttons, forms, cards, and grids, which help in building a clean and consistent UI faster.

---

## 🧠 OOPs Concepts

This project follows basic **Object-Oriented Programming (OOPs)** principles such as:

- **Modularity:** Code is divided into components, routes, controllers, and models  
- **Reusability:** React components and backend functions are reused  
- **Separation of Concerns:** Frontend, backend, and database logic are separated  
- **Maintainability:** The structured approach makes the code easier to manage and update
